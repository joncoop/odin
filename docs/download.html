The repository contains a dist folder which has the files you need to get started. The framework is defined entirely in odin.css. The main.css file just contains a somewhat organized template for adding any custom CSS or applying overrides to Odin's default styling.

  
    dist/
    ├── css/
    │   ├── main.css
    │   └── odin.css
    ├── js/
    │   └── odin.js
    └── index.html

The fastest way to get started is simply to make a copy of the dist folder and start working.

If you are integrating Odin into an existing project, then copy odin.css into your project directory. Any resets such as normalize.css should be linked before odin.css. Custom stylesheets and any stylesheets that override default Odin styling should be linked after.

The responsive navbar element and carousel require odin.js. Both are dependent on JQuery (not included).
